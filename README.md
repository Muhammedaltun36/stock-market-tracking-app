# 📈 Borsa Takip Uygulaması

Bu proje, Yazılım Mühendisliği dersi kapsamında, finansal verileri anlık veya dinamik olarak takip etmek, kullanıcı portföyünü yönetmek ve analiz etmek amacıyla geliştirilmiştir. Yazılım yaşam döngüsü ve nesne yönelimli programlama prensiplerine uygun olarak tasarlanmıştır.

## 🚀 Özellikler
* **Dinamik Takip:** Hisse senetleri ve borsa verilerinin anlık/güncel takibi.
* **Portföy Yönetimi:** Kullanıcıların sanal cüzdan oluşturarak varlıklarını izleyebilmesi.
* **Yazılım Mimarisi:** Temiz kod (Clean Code) ve esnek genişletilebilir mimari yapısı.

## 🛠️ Kullanılan Teknolojiler
* **Dil:** Java
* **Kütüphaneler/Teknolojiler:** JavaFX / Swing , SQL
